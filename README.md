# PEI Budget Tracker

An interactive budget tracker for Prince Edward Island residents to explore provincial spending, revenue, debt sustainability, and fiscal policy.

## Features

- **Overview**: Fiscal summary with key metrics and deficit tracking
- **Spending**: Detailed breakdown by department and category
- **Revenue**: Revenue sources and variance analysis
- **Capital**: Capital project investments
- **Economy**: Economic indicators and sector performance
- **Demographics**: Population trends, projections, immigration, and labour force data
- **Trade**: Export data, US dependency analysis, and tariff risk scenarios
- **Federal Transfers**: Federal transfer dependency and provincial comparison
- **Debt**: Debt sustainability trajectory and interest cost analysis
- **Tax Policy**: HST impact calculator and business competitiveness factors
- **Analyst Views**: Credit ratings from DBRS, Moody's, and analyst commentary

## Data Sources

- PEI Department of Finance
- PEI Statistics Bureau (51st Annual Statistical Review 2024)
- Statistics Canada Population Projections
- TD Economics, RBC Economics, DBRS Morningstar

## Running Locally

```bash
npm install
npm start
```
